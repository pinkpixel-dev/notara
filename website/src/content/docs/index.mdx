## What Notara includes

<CardGrid stagger>
  <Card title="Markdown notes">
    Rich note editing, live preview, a formatting toolbar, GitHub-flavored
    tables, syntax-highlighted code blocks, tags, pinning, and save shortcuts.
  </Card>
  <Card title="Todos and planning">
    Date-based task lists, sub-items, progress tracking, time scheduling, and a
    calendar view with upcoming and selected-day tabs.
  </Card>
  <Card title="Vision boards">
    Drag, resize, recolor, and filter mixed image and text boards for planning,
    inspiration, and reference gathering.
  </Card>
  <Card title="AI assistant">
    Pollinations-backed text and image generation with configurable models, chat
    persistence, and save flows into notes or boards.
  </Card>
  <Card title="Local-first storage">
    Desktop builds automatically use a Notara workspace folder. Web builds fall
    back to browser storage when filesystem access is unavailable.
  </Card>
  <Card title="Optional legacy extras">
    Supabase auth and GitHub, Google Drive, and Dropbox integrations remain
    available as opt-in, legacy-oriented capabilities.
  </Card>
</CardGrid>

## A quick orientation

Notara’s current release defaults to the **midnight** theme with the **pink** accent selected. The
same release also introduced app-wide glass styling, better AI persistence, richer markdown preview,
desktop installers, Docker delivery, and automatic desktop workspace storage. If you want the full
breakdown, head to the [release notes](/reference/releases/).
